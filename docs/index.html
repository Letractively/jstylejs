jStyle, An Easy-to-Use JavaScript Based Advanced CSS Render Engine.

jStyle is a fast and easy-to-use JavaScript Engine that make you can easily to develope web applications which have advanced UI and logic. It is also easy to extend by other developers.
jStyle is designed to help you rapidly develope web applications.

  • Easy-To-Use

    With jStyle,you can easy to develope web applications which have advanced UI and logic.

  • CSS3 Compliant

    jStyle uses Sizzle as the engine of css selectors. So it supports CSS 3 selectors!

  • Cross-browser

    IE 6.0+, FF 2+, Safari 3.0+, Opera 9.0+, Chrome
