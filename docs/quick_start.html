classes

This style is use to set the specified elemenet(s)'s className attribute.
There are four operators：
"+" means add classes
"-" means delete classes
">" means toggle classes
"=" means set the classes
For example:
A Div

manipulate

This style is use to manipulate the specified elemenet(s).
There are five operators：
"-" means delete content
"+<" means add content at beforeBegin
">+" means add content at afterEnd
"+>" means add content at beforeEnd
"<+" means add content at afterBegin
For example:

effects

This style is use to make the effects of the element(s).
There are three operators by this time：
"+" means show the element(s).
"-" means hidden the element(s).
">" means toggle the element(s).
