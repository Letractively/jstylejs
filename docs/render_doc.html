Rule Reference

You can just write the html Element's attribute of ”jstyle“ to setup the Render Rule。 For example:
<input jstyle="render:{dialog:{url:'get_input.php?value=xxx'}}" title=" Text Element" name="elementText" id="elementText" type="text" value="" />
Rule Parameter Description Example
dialog Object Open a dialog to get the element's value.
  parameters
  title: the tile of dialog button.
  text: the element's text.
features:the dialog's features.@see sFeatures of window.showModalDialog 		dialog:{url:'get_input.php?value=xxx',
features:'dialogWidth:300px;dialogHeight:600px'}
validation Object Validation the element's value from the giving url. Using ajax.
  parameters
  title:the tile of validation button. 		validation:{url:'validation_input.php?value=xxx',
title:' Validate if the user exists.'}
getvalue Object Get the element's value from the giving url.Using ajax.
  parameters
  title:the tile of getvalue button.
readonly:is the element readonly or not? 		getvalue:{url:'get_input.php?value=xxx',
title:'get the element's value',readonly:true}
