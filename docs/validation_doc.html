Rule Reference

You can just write the html Element's attribute of ”jstyle“ to setup the Validation Rule。 For example :
<input jstyle="validation_rule:{name:' Text Element',required:true,min_length:5,max_length:8}" title=" Text Element" name="elementText" id="elementText" type="text" value="" />
Rule Parameter Description Example
name String The element's name, use to alert message to users. name:'Name'
{parameter_name}_message String when {parameter_name} validation failed, show this message to user. email_message:'the value should be email '
required boolean Returns FALSE if the form element is empty. required:true
matches Array of String Returns FALSE if the form element does not match the one in the parameter. matches:['form_item']
min_length integer Returns FALSE if the form element is shorter then the parameter value. min_length:6
max_length integer Returns FALSE if the form element is longer then the parameter value. max_length:12
littler string Returns FALSE if the form element is littler then the parameter value. littler:'455666'
greater integer string FALSE if the form element is greater then the parameter value. greater:'455666'
exact_length integer Returns FALSE if the form element is not exactly the parameter value. exact_length:8
alpha boolean Returns FALSE if the form element contains anything other than alphabetical characters.
alpha_numeric boolean Returns FALSE if the form element contains anything other than alpha-numeric characters.
alpha_dash boolean Returns FALSE if the form element contains anything other than alpha-numeric characters, underscores or dashes.
numeric boolean Returns FALSE if the form element contains anything other than numeric characters.  
integer boolean Returns FALSE if the form element contains anything other than an integer.  
email boolean Returns FALSE if the form element does not contain a valid email address.  
ip boolean Returns FALSE if the supplied IP is not valid.  
base64 boolean Returns FALSE if the supplied string contains anything other than valid Base64 characters.  
regexp String Returns False if don't match the giving regexp string。 regexp:'^([a-z0-9\+_\-]+)(\.[a-z0-9\+_\-]+)*@([a-z0-9\-]+\.)+[a-z]{2,6}$'
